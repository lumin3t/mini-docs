# **Project: Collaborative Text Editor (mini-docs)**

This is a real-time editor built for a **Hackathon**. It works like Google Docs, allowing multiple people to edit the same page at once without lag or overwriting each other.

### **The Tech Stack**

- **Framework:** Next.js
- **Editor Engine:** Tiptap
- **Real-time & Presence:** Liveblocks
- **Styling:** Tailwind CSS
- **Database:** MongoDB
    
### **Main Features**

- **Live Presence:** See who else is currently in the document.
- **Cursor Tracking:** Watch other users' cursors and selections move in real-time.
- **Auto-Save:** Everything is automatically synced to MongoDB.
- **History:** Track changes and jump back to previous versions.
    
### **How to run it**

1. **Clone & Enter:**
    ```bash
    git clone https://github.com/lumin3t/mini-docs.git && cd frontend
    ```
    
2. **Install:**
    ```bash
    npm install --legacy-peer-deps
    ```
    
3. **Start:**    
    ``` bash
    npm run dev
    ```

    (AI Tools like ChatGPT and Gemini used for automation)
